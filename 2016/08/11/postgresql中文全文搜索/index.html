创建索引

下面假设你已经安装并配置好了zhparser.

zhparser.multi_short = on;设置为全局级别后再创建索引.

alter role all set zhparser.multi_short=on;

创建索引:

create index idx_name on ts using gin(to_tsvector('testzhcfg', name));
