Netty粘包和拆包资料收集

2018-05-03 168 words 1 min read times read

Contents

  • 参考资料

维基百科对 TCP 的定义为:

传输控制协议（英语：Transmission Control Protocol，缩写为 TCP）是一种面向连接的、可靠的、基于字节流的传输层通信协议，由IETF的RFC 793定义.

说白了, 所谓的粘包和拆包, 只是对TCP的字节流数据进行应用层的处理问题.比如 HTTP 协议, 等. 如果是自己定制协议, 则要考虑好协议的数据格式问题~
