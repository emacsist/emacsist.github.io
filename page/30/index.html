title: “使用GDB调试Redis” date: 2019-08-09T11:36:00+08:00 draft: false tags: redis gdb 环境准备 以 Redis 4.0.14 为例子. Linux 环境 下载 redis 源码 redis-4.0.14.tar.gz 编译时使用 make noopt 然后启动 redis-server 安装 GDB : sudo apt-get install gdb 禁用 ptrace : sudo echo 0 > /proc/sys/kernel/yama/ptrace_scope 调试 gdb ./src/redis-server # 在 main 函数中打断点 (gdb) b main # 执行. 这时可以发现 gdb 停留在了 main 函数入口处.
