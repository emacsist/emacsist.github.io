jetty.home 和 jetty.base

可以看作： ${jetty.home} 是jetty的安装目录 ${jetty.base} 是实例目录

Jetty常用命令

查看帮助

java -jar ${jetty.home}/start.jar --help

指定jetty.home和jetty.base

java -jar /home/yang/Java/jetty-distribution-9.2.13.v20150730/start.jar \
jetty.home=/home/yang/Java/jetty-distribution-9.2.13.v20150730 \
jetty.base=/home/yang/Java/jetty-distribution-9.2.13.v20150730/demo-base

列出配置

cd $
java -jar ${jetty.home}/start.jar --list-config

创建一个实例

cd /path/to/jetty/base/empty/dir

java -jar ${jetty.home}/start.jar --add-to-startd=http,deploy

添加https支持
--add-to-startd=https

Jetty配置说明

在webapps下面，如果有一个xxx.war文件，并且有一个xxx.xml文件，则以xxx.xml为准来部署xxx.war应用的上下文.xxx.d目录，则是用于被xxx.xml使用，用来注入到xxx.war的额外配置.
