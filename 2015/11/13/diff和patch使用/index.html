diff

文件判断

三个减号(---)：原始文件 三个加号(+++)：目标文件

内容

-号开始的行：只出现在原始文件中的行 +号开始的行：只出现在目标文件中的行 以空格开始的行：同时出现在原始文件和目标文件中

差异

注意：diff 是用于行比较的.

前后用两个@@标识来定位差异语句的开始。例如：

@@ -1,4 +1,4 @@：

-1,4：表示原始文件，从第一行开始的4行（注意，不是到第四行） +1,4：表示目标文件，从第一行开始的4行（注意，不是到第四行）

patch

f1, f2, diff.txt

假设 diff.txt 是由 diff -u f1 f2 的结果.

删除f2，想要还原

cp f1 f2
patch f2 < diff.txt

删除f1，想要还原

cp f2 f1
patch -R f1 < diff.txt

注意，有参数-R
