Prelude

它是一个高度可配置的开箱即用的Emacs配置文件集合.

  • Github Prelude

开启里面的modules

默认情况下, 安装好 prelude 是没有开启它的模块功能的,要开启自己想要的模块, 可以按以下做:

cd ~/.eamcs.d/
cp sample/prelude-modules.el .
emacs prelude-modules.el

然后按里面的模块, 根据自己的需要, 去开启或者关闭.

自定义自己的额外配置

将自己的配置文件放到 ~/.emacs.d/personal/preload/ 目录下即可.

例如, 我有一个配置文件 ~/.emacs.d/personal/preload/yang.el , 这样子 prelude 就会自动加载这个配置文件了.

我刚开始做 prelude , 我只是暂时添加了自己喜欢的主题, yang.el 的内容就只有以下一行:

(setq prelude-theme 'solarized-dark)

注意, 默认情况下, prelude 并没有安装这个主题, 所以先要安装下这个主题先:

M-x package-install RET solarized-theme

注意

为了升级方便, 其他地方的文件自己不要随便添加或者修改, 最好按 prelude 的约定来做:

  • ~/.emacs.d/prelude-modules.el : 这里开启或关闭自己需要的模块.

  • ~/.emacs.d/personal/preload/ : 这个目录,存放自定义的配置文件.
