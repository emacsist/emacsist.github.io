部署到 Github Pages

参考: hexo-deploy-git

注意,在执行以下命令时, 可能要等上好一会儿, 当时自己一直以为是卡住了, 或者是出问题,所以常常按 Ctrl+C 来中断. 其实它还在执行中的.

npm install hexo-deployer-git --save

所以,最好添加上详细输出的参数,看来整个执行的过程:

npm install hexo-deployer-git --save --verbose
